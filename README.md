# create-mf2-app

The easiest way to start a new project with the MF2 Stack.

## Getting Started

```bash
npm create mf2-app@latest
# or
yarn create mf2-app
# or
pnpm create mf2-app
```

## CLI Options

```bash
# Specify a project name
npm create mf2-app@latest my-app

# Use a specific package manager
npm create mf2-app@latest my-app --use-pnpm
```
