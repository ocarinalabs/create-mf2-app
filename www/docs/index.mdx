## Learn more

MF2 is packed with features:

- **All the tech** - Next.js 15, TypeScript, Tailwind, shadcn/ui
- **Auth & Payments** - Clerk for authentication, Polar for payments
- **Real-time data** - Convex for instant updates and sync
- **Email & Analytics** - Resend for emails, PostHog for analytics
- **AI-optimized** - Built for Cursor, Claude Code, and Copilot
- **Production ready** - Deploy to Vercel in one click
