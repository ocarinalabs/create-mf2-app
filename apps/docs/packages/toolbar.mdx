title: Toolbar
description: Vercel Toolbar for development and preview environments.
---

The Vercel Toolbar appears in development and preview deployments with feature flag overrides, draft mode, visual editing, and deployment comments.

`@repo/next-config` enables the toolbar automatically. No additional setup required.

See [Vercel Toolbar docs](https://vercel.com/docs/workflow-collaboration/vercel-toolbar).
